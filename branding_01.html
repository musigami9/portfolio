Branding and Identity: Project 01

Thinking Inside the Box

We will focus on learning to essentialize uncommon figures using a set of predetermined constraints. The constraints in this project present challenges that should engage us with a set of diverse, modern approaches to creating simple forms.

Luchador | full figure shown, straight-on front perspective, limited to two colors, must use only shape.

Lancia stratos | must be side profile, unlimited color, using both uniform line and shape.

Orangutan | only the orangutan's face may be depicted, limited to black and white, using only lines.
